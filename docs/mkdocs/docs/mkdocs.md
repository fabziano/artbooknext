# 2. Instalação do MkDocs e do Tema Material

Com o Python já configurado e no PATH, abra o Prompt de Comando e use o **pip** para instalar o gerador de sites MkDocs e o tema Material.

## Comando de Instalação

`pip install mkdocs mkdocs-material`

## Verificação

Para confirmar que a ferramenta está disponível globalmente:

`mkdocs --version`
