# 3. Estrutura Inicial do Projeto

O primeiro passo para criar a documentação é gerar a estrutura de pastas básica.

## Criação do Projeto

No terminal, execute:

`mkdocs new NOME_DO_SEU_PROJETO` 

Isso cria pasta com o nome do projeto contendo o arquivo de configuração e a pasta **docs/**.

## Acessando a Pasta

Em seguida, entre no diretório do novo projeto para continuar a configuração:

`cd NOME_DO_SEU_PROJETO`
