# 1. Instalação do Python

O **Python** é o pré-requisito para o MkDocs, pois ele inclui o gerenciador de pacotes **pip**, que é o que usaremos para instalar o MkDocs.

## Detalhes da Instalação (Windows)

1 - Baixe o instalador do Python pelo site oficial.

2 - Na primeira tela de instalação, marque a caixa:

`Add python.exe to PATH`

> Isso permite usar o **pip** e outros comandos do Python diretamente no Prompt de Comando, de qualquer diretório.

3 -  Após a conclusão, clique em **Disable path length limit** (desabilitar o limite de tamanho do caminho) para evitar problemas futuros com bibliotecas complexas.
